# e世代生命密碼分析儀

一個以數字收斂與原型敘事為基礎的繁體中文自我反思工具。

## 公開網站

https://endfew-ai.github.io/e-shidai-life-code/

## 隱私與使用方式

- 所有輸入只在使用者自己的瀏覽器內計算，不會送到伺服器。
- 建議使用生日、手機末碼或自訂數字，不要輸入完整身分證號。
- 內容屬文化娛樂與自我反思用途，不是醫療、心理或其他專業判斷。

## 本機開啟

一般使用可直接開啟根目錄的 `index.html`。開發版本可使用：

```powershell
npm ci
npm run dev
```

完整驗證：

```powershell
npm test
```

## 發布

GitHub Pages 使用 `main` 分支根目錄作為公開來源；推送更新後，GitHub 會自動重新發布。
