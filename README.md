# e世代生命密碼分析儀

以「生日命碼」為主功能的繁體中文數理分析工具，另提供數字頻譜與三數取卦補充功能。所有結果都顯示固定規則與可核對算式。

## 公開網站

https://endfew-ai.github.io/e-shidai-life-code/

同一個網址會依螢幕寬度自動適配手機與電腦，不需要切換版本。

## 視覺方向

- 採 B「玄星觀象」：深靛黑、古金、精密星軌與宋明書卷感。
- 主要視覺與分析模組使用專案專屬生成圖，不把易經符號當成首頁主題。
- 生日命碼是第一層入口，數字頻譜為次要功能，三數取卦與《周易》本文為補充資料。

## 隱私與使用方式

- 所有輸入只在使用者自己的瀏覽器內計算，不會送到伺服器。
- 生日模式只需要西元出生日期，不需要姓名、時辰或身分證號。
- 自訂數字模式建議只使用手機末碼、門牌或非敏感號碼，不要輸入完整身分證、金融帳號。
- 三數取卦只做本卦、互卦、動爻與變卦的固定計算；附 64 卦古籍本文，不提供 AI 解卦或吉凶判斷。
- 內容屬文化娛樂與自我反思用途，不是醫療、心理或其他專業判斷。

## 採用規則

- 生日命碼：月、日、年分段化簡；生命路徑與生日核心保留 11、22、33，態度數及個人流年化簡至 1～9。
- 九宮分布：只統計 1～9 出現次數並採洛書版位呈現，不宣稱是傳統命盤。
- 三數取卦：第一數取上卦、第二數取下卦、第三數取動爻；使用固定八卦與六十四卦表。

## 本機開啟

因靜態版使用 ES Module，請透過本機 HTTP 伺服器預覽，不要直接用 `file://` 開啟 `index.html`：

```powershell
npx --yes serve .
```

開發版本可使用：

```powershell
npm ci
npm run dev
```

完整驗證：

```powershell
npm test
```
