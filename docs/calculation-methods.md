# 梅花易數演算方法

版本：`kangjie-calculation-trace-v1`

## 共用規則

- 六爻由初爻至上爻儲存。
- 下卦為第 1 至 3 爻，上卦為第 4 至 6 爻。
- 卦數用 `mod1(total, 8)`，整除時為 8。
- 動爻用 `mod1(total, 6)`，整除時為 6。
- 互卦下卦取本卦第 2、3、4 爻。
- 互卦上卦取本卦第 3、4、5 爻。
- 動爻在第 1 至 3 爻時，下卦為用、上卦為體。
- 動爻在第 4 至 6 爻時，上卦為用、下卦為體。

## 年月日時

```text
上卦數 = 年支數 + 農曆月 + 農曆日
下卦數 = 年支數 + 農曆月 + 農曆日 + 時支數
動爻數 = 年支數 + 農曆月 + 農曆日 + 時支數
```

十二地支依序為子 1 至亥 12。

## 物數

```text
上卦數 = 可數物件數
下卦數 = 時支數
動爻數 = 可數物件數 + 時支數
```

## 聲音

### 單一聲數

```text
上卦數 = 聲數
下卦數 = 聲數 + 時支數
動爻數 = 聲數 + 時支數
```

這是對傳本文字「數得幾數，起作上卦，加時數配作下卦」的明示實作解讀，結果會保留該假設。

### 分段聲數

```text
上卦數 = 第一段聲數
下卦數 = 第二段聲數
動爻數 = 第一段聲數 + 第二段聲數 + 時支數
```

## 字占

### 一字

楷書左部陽畫作上卦，右部陰畫作下卦，左右合計取動爻。Unicode 總筆畫無法自動拆成左右陰陽畫，因此必須人工確認。

### 二至三字

依每字總筆畫分組。偶數字數平分；奇數字數少一字在上、多一字在下。各組筆畫合計取上下卦，全部筆畫合計取動爻。

### 四至十字

古籍主法採人工確認的平、上、去、入，依序為 1、2、3、4。不可用現代國語四聲自動冒充。另提供明確標示的使用者筆畫分組模式。

### 十一至一百字

只計漢字。偶數平分；奇數少一字在上、多一字在下。上下字數合計取動爻。

## 丈尺與尺寸

丈尺占：

```text
上卦數 = 丈數
下卦數 = 尺數
動爻數 = 丈數 + 尺數
寸數不用
```

尺寸占：

```text
上卦數 = 尺數
下卦數 = 寸數
可核傳本主法動爻 = 尺數 + 寸數 + 時支數
未證流傳異法動爻 = 尺數 + 寸數
```

目前可核的《梅花易數》轉錄與影本均支持尺寸合數加時取爻；不加時只為相容需求保留，介面與結果不得把它冒充為已證古本。

## 後天端法

```text
上卦 = 使用者確認的物象卦
下卦 = 使用者確認的方位卦
動爻數 = 物象卦數 + 方位卦數 + 時支數
```

為人、自己、動物、靜物及方位入口共用此引擎，但保留不同 scenario。物象文字只回傳候選，不自動宣稱唯一對應。動物及靜物若未輸入觸發事件會顯示「不動不占、無故不占」警告。

## 姓名或姓氏加數

```text
加數 = 所輸入姓名或姓氏的逐字筆畫合計
上卦數 = 年支 + 農曆月 + 農曆日 + 加數
下卦數 = 上卦數 + 時支
動爻數 = 下卦數
```

每個字的筆畫、來源、資料版本及人工覆寫狀態都寫入演算追蹤。本次核對卷一未找到足以支持此單一公式的原文，所以此入口標為流傳／使用者指定規約，不冒充古籍主法。
