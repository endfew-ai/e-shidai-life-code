# 生命靈數測試矩陣

版本：2.0.0
日期：2026-07-23

## 一、測試原則

- 純演算法使用固定 `todayValue`、`currentYear`、`createdAt`，不得依機器當下時區漂移。
- 新版預設與 `legacy-project-v1` 分開驗證。
- 民俗含義不測「真假」，只測資料表、規則選擇、輸出一致性與警告。
- 正式驗收必須涵蓋單元、整合、瀏覽器、手機版、隱私及 build 輸出。
- 本文件是應測矩陣；實際是否通過以測試命令與發布驗證紀錄為準，不因列在表中就算已測。
